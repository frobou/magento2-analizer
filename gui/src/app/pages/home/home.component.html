O Magento Analyser é uma ferramenta elaborada para auxiliar no trabalho dos desenvolvedores Magento, agilizando a busca de Plugin e Observer contidos em um projeto.


Com uma simples busca, você irá obter uma lista de todos os Plugins ou Observers existentes em um projeto, ainda sendo possível refinar essa busca, procurando apenas o que seja de sua necessidade.


Com um único clique, você conseguirá abrir o arquivo, na linha que contém o artefato desejado, sendo que as IDEs disponibilizas para isso são, o Visual Studio Code e o PHP Storm.
